# FAQ ❓

**Q: How do I reset my password?**  
A: Go to **Profile → Change Password** and enter a new password.  

**Q: Can I use the dashboard on multiple servers?**  
A: Yes, add additional server credentials in **Settings → Servers**.  

**Q: How do I enable notifications?**  
A: Go to **Settings → Notifications** and enable alerts for the desired events.  

**Q: Where can I get support?**  
A: Visit the **Support** page in the dashboard or contact your admin team.  

**Q: How do I change the language or theme?**  
A: Use the **Settings** panel to switch between English and German or to choose your preferred theme.  

**Q: How do I back up my configuration?**  
A: Use **Backup & Restore** in the dashboard to create versioned backups of all settings.  

**Q: How do I restore a previous backup?**  
A: Open **Backup & Restore**, select a backup version, and click **Restore**.  

**Q: Who can manage users and permissions?**  
A: Only users with **Admin** or **Owner** roles can manage roles and permissions.
